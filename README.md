# Nemo — красивый шаблон сайта на React + Vite

Минималистичный, адаптивный шаблон для быстрой разработки.

Запуск локально (Windows PowerShell):

```powershell
npm install
npm run dev
```

Откройте в браузере URL, который покажет `vite` (обычно `http://localhost:5173`).

Если хотите, могу сейчас запустить `npm install` и `npm run dev` в терминале — скажите, разрешаете ли вы установку зависимостей.
